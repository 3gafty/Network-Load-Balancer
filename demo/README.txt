Для работы примера нуже gnome-terminal либо прописать свой терминал в
запускающем скрипте run.sh.
Три клиента принимают посылки через балансировщик от источника
Запустить скрипт ./run.sh и наблюдать

При сборке использовался следующий конфиг
{
	"num_of_msg_per_sec": 1,

		"port": 12345
	},
	"destinations":

		{
